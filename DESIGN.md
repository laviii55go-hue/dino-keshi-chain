## ゲームルール

### 盤面
- **5列 × 7行**
- 6種類の恐竜をランダム配置（初期状態でマッチなし保証）

### 恐竜キャラ（6種）

| スロット | 画像 | 恐竜 | 分類 |
|---------|------|------|------|
| 0 | dino-0.png | ティラノサウルス | 肉食 |
| 1 | dino-4.png | ステゴサウルス | 草食 |
| 2 | dino-2.png | プテラノドン | 飛行 |
| 3 | dino-3.png | トリケラトプス | 草食 |
| 4 | dino-5.png | スピノサウルス | 肉食 |
| 5 | dino-7.png | パキケファロサウルス | 草食 |

### 操作
- **タップ2回**: 1つ目を選択（青枠）→ 2つ目をタップ（赤枠）で入れ替え（距離制限なし）
- **スワイプ**: 隣接セルとの入れ替え
- **同じセル再タップ**: 選択解除
- マッチしない入れ替えも**回数を消費して確定**（戦略的配置換え可能）

### 消えるルール

| パターン | 消える？ |
|---------|---------|
| 横3つ以上（直線） | ✅ |
| 縦3つ以上（直線） | ✅ |
| 隣接4つ以上（形は問わない） | ✅（2×2、L字、T字、Z字 等） |
| 直線でない3つ | ❌ |
| 2つだけ | ❌ |
| 斜め | ❌ |

- 直線マッチのセルに隣接する同種セルはflood-fillで統合
- 消去 → 落下（ポップインアニメーション） → 補充 → 再判定（連鎖）を繰り返す

## 岩ブロック（難易度スケーリング・2026/04/24 追加）

### スポーン
- **序盤〜中盤（Lv3, 6, 9, 12）**: 3レベルごとに **+3個**
- **後半（Lv15以降）**: **毎レベル +3個**（Lv15, 16, 17, 18, 19, 20...）
  - 理由：Lv11以降は毎Lvに1,500点必要で時間がかかるため、岩の出現ペースを上げて緊張感維持
- **自然落下演出**: マッチ後の空きセルを上から補充するタイミングで、新規落下セルの一部が岩として降ってくる
- 上限なし（盤面が埋まる限り追加・現状では削除ペースが上回る設計）

### 耐久（HP）
| レベル帯 | 新規岩のHP |
|---------|---|
| Lv1-10 | ♥1 |
| Lv11-20 | ♥2 |
| Lv21+ | ♥3 |

### 挙動
- **マッチ非参加**: 岩は恐竜と一致しないのでマッチグループに入らない
- **入れ替え不可**: タップ・スワイプどちらも反応しない
- **落下**: 重力で落ちる（通常セルと同じ）
- **ダメージ**: 隣接セルがマッチ消去されるたびに **HP -1**
- **破壊時得点**: **+30点**

### スキルとの相性（2026/04/24 更新: 範囲系は即破壊 → -1ダメージに変更）

| スキル | 岩への効果 |
|---|---|
| ティラノ（草食全消し） | ❌ 岩は対象外 |
| ステゴ（種類指定全消し） | ❌ 岩は対象外 |
| プテラ（ランダム10） | 🔨 **-1 ダメージ**（隣接マッチと同じ・HP>0なら残存） |
| トリケラ（横一列） | 🔨 **-1 ダメージ** |
| スピノ（周囲8マス） | 🔨 **-1 ダメージ** |
| パキケファロ（縦+最下段周囲） | 🔨 **-1 ダメージ** |

→ Lv11+ で ♥2 の岩が出るようになると、スキル1発では壊れない（2回必要）。
後半の難易度上昇をスキル強化ではなく「継続耐久」で演出する設計。

## スキルシステム（独自要素）

### 獲得条件（2026/04/24 案B 採用・恐竜ごとに差別化）

| 恐竜 | 必要個数 | 備考 |
|---|:-:|---|
| トリケラ | **5個** | 弱（横一列・確定5マス） |
| ステゴ | **5個** | 中弱（種類指定・5-7マス・岩不可） |
| プテラ | **6個** | 中（ランダム10マス） |
| スピノ | **6個** | 中（周囲8マス・確定9マス） |
| パキケファロ | **6個** | 中〜強（縦一直線+着弾周囲 ≒ 12マス） |
| ティラノ | **8個** | 最強（草食全消し ≒ 10-13マス） |

同時消去数が該当恐竜の閾値に達すると、その恐竜のスキルがストックに追加される。

### ストック
- 画面左側に縦1列（最大**9個**）
- 下から積み上がる
- 溢れたら古いものから消える
- タップで発動（回数を消費しない）
- **2026/04/24 追加**: スキル発動ごとに残数 **+1回** 付与（溜め込まず使うほど得な設計）

### スキル一覧

| 恐竜 | スキル名 | 発動方法 | 効果 |
|------|---------|---------|------|
| ティラノ | 捕食 | 即発動 | 草食恐竜（ステゴ・トリケラ・パキケファロ）を全消し |
| ステゴ | 尻尾スイープ | ターゲット指定 | 指定した種類の恐竜を全消し |
| プテラ | 空爆 | 即発動 | ランダム10個を爆破（爆発エフェクト） |
| トリケラ | 突進 | ターゲット指定 | 横一列消し |
| スピノ | 水撃 | ターゲット指定 | 指定セル＋周囲8マス消し |
| パキケファロ | 頭突き | ターゲット指定 | 縦一直線＋最下段で周囲5マス消し |

### 発動フロー
1. ストックの恐竜をタップ
2. カットシーン: 左からスライドイン → 恐竜名 + スキル説明表示 → 右へスライドアウト
3. ターゲット指定系: タッチしている間に影響範囲をプレビュー（赤枠＋赤背景）→ 離して発動
4. 効果発動 → 消去 → 連鎖チェック

### 手数0時のスキル
- 手数0でスキルが残っていればゲーム続行
- 通常の入れ替え操作は不可（「スキルを使用してください」メッセージ）
- スキル発動 → 連鎖 → レベルアップで手数回復の可能性あり

## データ保存（AsyncStorage）

| キー | 内容 |
|------|------|
| dino_chain_game_state | 盤面・スコア・手数・レベル・スキルストック |
| dino_chain_ranking | ローカルTOP10ランキング配列（オフライン・マイ記録） |
| dino_chain_high_score | ハイスコア |
| dino_chain_player_name | プレイヤー名 |
| dino_chain_settings | 設定（自動登録ON/OFF） |
| dinoChainLastLogin | グローバルランキング連続ログイン日（最終ログイン日） |
| dinoChainConsecutiveDays | 連続ログイン日数 |

## データ保存（Firebase Realtime Database）

- **URL**: `https://dino-keshi-chain-default-rtdb.asia-southeast1.firebasedatabase.app`（連鎖専用プロジェクト・本体とは独立）
- **パス**: `/scores.json`
- **フィールド**: name / score / level / date / ts / consecutiveDays / lastLoginDate
- **SDK**: 使わず `fetch` で REST API 直接呼び出し（Expo Go 互換）
- **書き込み**: ゲームオーバー時に `submitGlobalScore(name, score, level)` で POST
- **読み込み**: `fetchGlobalRankings(period)` で月次取得→クライアントで日次/週次フィルタ
- **キャッシュ**: 60秒 in-memory
- **ルール設定例**（ロックモード作成後に手動設定）:
  ```json
  {
    "rules": {
      "scores": {
        ".read": true,
        ".write": true,
        ".indexOn": ["ts"]
      }
    }
  }
  ```
